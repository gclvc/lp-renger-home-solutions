# Renger Home Solutions

Angular landing page for a San Francisco and Bay Area handyman and property maintenance business.

## Run locally

This project was generated with Angular 21, which requires Node 20.19+ or Node 24+.

```powershell
npm install
npm start
```

If the global `node` on the machine is still Node 18, switch Node versions first with nvm or run Angular using a newer Node executable.

## Pages

- `/` Home landing page with fixed CTA, hero estimate card, services, trust sections, FAQ, and CTA band.
- `/about` Company positioning page.
- `/service-areas` Local SEO/service-area page for San Francisco and Bay Area locations.
- `/contact` Contact page with an Angular form.

## Contact form

The contact form currently captures the submission in the UI as a demo. To send real emails without opening the visitor's email app, connect the form to a service such as Formspree, Web3Forms, EmailJS, Netlify Forms, or a small API endpoint.

## Placeholder content

Phone, address, email, insurance details, warranty language, pricing, and service copy should be confirmed with the business before publishing.
